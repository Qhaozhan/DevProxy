使用流程

  1. 添加路由 — 填写需要代理的域名和对应的上游地址
  2. 保存配置 — 点击「保存配置」将路由规则写入磁盘
  3. 生成证书 — 为域名生成 TLS 证书并安装到系统信任库
  4. 写入 hosts — 将域名指向本地（需管理员权限）
  5. 启动代理 — 开始监听并转发 HTTPS 请求
  6. 正常使用 — 此时请求该域名会经由本地代理转发至上游
  7. 停止时 — 依次点击「停止代理」→「恢复 hosts」即可还原
